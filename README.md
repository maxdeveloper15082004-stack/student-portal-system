# Student Portal

A Django-based Student Portal application.

## Project Structure

- `backend/`: Django project and applications
- `frontend/`: HTML Templates and Static assets

## Setup

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run migrations:

   ```
   cd backend
   python manage.py migrate
   ```

3. Run server:
   ```
   python manage.py runserver
   ```
